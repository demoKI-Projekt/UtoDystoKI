# UtoDystoKI
Ein Coin Flip Geschichten Spiel

Künstliche Intelligenz wird schon und zukünftig noch mehr in den unterschiedlichsten Bereichen eingesetzt. Ob das gut oder schlecht ist, wird sich wahrscheinlich noch zeigen. Man könnte sich aber schon einmal Gedanken über eine mögliche DystoKI oder UtoKI machen.

Man sucht sich eine Einsatzmöglichkeit (z.B. Medizin) aus den Bildern rechts aus. Danach flippt man die Münze. Im Anschluss spinnt man dann zu der Einsatzmöglichkeit eine Dysto- oder eine UtoKI, je nachdem, was die Münze anzeigt.
